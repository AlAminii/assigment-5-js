What is the Document Object Model (DOM)?

The Document Object Model (DOM) is a programming interface for web documents. It represents the structure of a web page as a tree of objects, allowing developers to access, modify, and manipulate elements, attributes, and content dynamically. The DOM acts as a bridge between HTML, CSS, and JavaScript, enabling the creation of interactive and dynamic websites. For example, using JavaScript, you can change the text of an HTML element, update styles, or respond to user events like clicks or key presses.

What is event delegation in the context of the DOM, and why is it useful?

Event delegation is a technique in JavaScript where you add a single event listener to a parent element to manage events for its child elements. Instead of attaching event listeners to each individual child element, the event is handled by the parent, leveraging the concept of event bubbling.
